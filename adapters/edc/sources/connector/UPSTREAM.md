# Upstream Source

This directory contains the generic EDC connector source used by the
Validation Environment.

- Upstream repository: `https://github.com/ProyectoPIONERA/EDC-asset-filter-dashboard`
- Upstream subdirectory: `asset-filter-template`
- Imported upstream commit: `9175ce552d93a0a06e464d077eb207a76f6d7f54`
- Import method: filtered subtree import preserving upstream commit authorship
  and dates for the retained connector files.

The import keeps the connector build files, Java source modules, runtime
resources, helper scripts, and connector documentation required to build and
operate the EDC connector inside this framework.

Generated artifacts were intentionally excluded from the imported history,
including Gradle caches, local build outputs, jars produced by the build,
logs, Python caches, and dashboard/UI-only documentation or generated files.

No standalone license file was present in the imported upstream subdirectory
at the commit above. Keep this attribution file with the imported source and
verify project-level licensing before redistributing the connector outside the
Validation Environment.
